# Pliocene rainfall and C4 evolution
Code for manuscript on Pliocene rainfall seasonality and C4 evolution over East Asia

- **data/** data used in this study
- **soil_water_isotope.R** calculate soil water isotope data 
- **JPI/** codes to perform the Bayesian inversion of the proxy system model for the triple oxygen isotope of pedogenic carbonates
- **JPI/BASS_bayes.R** proxy system model for the triple oxygen isotope of pedogenic carbonates 
- **JPI/BASS_driver.R** code to perform the Bayesian inversion
- **plot/** codes to plot figures
- **figures/** figures generated from the code
- **output/** calculated soil water isotope data and posterior estimates of the Bayesian inversion
